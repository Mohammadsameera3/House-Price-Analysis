# 🏡 House Price Analysis

## 📌 Project Overview
This project analyzes a house pricing dataset using Python, Pandas, Matplotlib, Seaborn, and SQL. The goal is to extract meaningful insights about factors influencing house prices.

---

## 📂 Dataset Information
The dataset contains 2000 records with the following features:

- Area
- Bedrooms
- Bathrooms
- Floors
- YearBuilt
- Location
- Condition
- Garage
- Price

---

## 🧰 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQLite (SQL Queries)

---

## 🔎 Project Workflow

### 1️⃣ Data Cleaning
- Checked duplicates
- Removed unnecessary columns
- Verified missing values

### 2️⃣ Exploratory Data Analysis (EDA)
- Distribution analysis
- Descriptive statistics
- Feature engineering (price per square unit)
- Group-by analysis

### 3️⃣ Data Visualization
- Histograms
- KDE plots
- Box plots
- Pie charts
- Bar charts
- Location-wise price comparison

### 4️⃣ SQL Analysis
- Converted dataset to SQLite database
- Executed filtering queries
- Extracted location-based insights

---

## 📊 Key Business Insights

- Suburban areas show highest average prices.
- Property area strongly influences overall price.
- 3–4 bedroom houses dominate market demand.
- Recently built houses command higher pricing.
- Location plays a crucial role in price variation.

---

## 🚀 How to Run This Project

1. Clone the repository
2. Install required libraries:
   ```
   pip install pandas numpy matplotlib seaborn
   ```
3. Open the Jupyter Notebook:
   ```
   jupyter notebook House-Price-Analysis.ipynb
   ```

---

## 🎯 Conclusion
House pricing is significantly influenced by location, area, and property configuration. This analysis demonstrates how Python and SQL can be used together for real estate data analysis.
